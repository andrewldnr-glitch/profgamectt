# ПрофМаршрут

**ПрофМаршрут** — русскоязычное статическое веб-приложение для профориентационных занятий со школьниками 13–17 лет. Проект подготовлен для использования в **ЦТТ Адмиралтейского района Санкт-Петербурга** и рассчитан на смешанный формат работы: телефон, компьютер педагога и проектор.

## Что внутри

Приложение работает как статический сайт и не требует backend. Внутри уже собраны пять игровых модулей:

1. **Угадай профессию** — карточки с 3–4 подсказками, ответом и коротким фактом.
2. **Правда или миф** — быстрые утверждения о профессиях и рынке труда с объяснением после ответа.
3. **Профессия по навыкам** — подбор профессии по набору навыков и контексту.
4. **Генератор профессий** — случайный выбор профессии для обсуждения в классе.
5. **Профессии будущего** — генератор новых ролей на стыке технологий и сфер.

Дополнительно есть **режим учителя**: запуск модулей, переход к следующему заданию, таймер, счёт команд, случайный выбор ученика или команды, переключение в режим проектора.

## Для кого это сделано

Проект собран как реальный учебный инструмент для профориентационной работы в учреждении дополнительного образования. Интерфейс выдержан в нейтральном, современном и неинфантильном стиле, а тексты рассчитаны на уважительный разговор с подростками.

## Как открыть локально

### Вариант 1. Самый простой

Откройте файл `index.html` двойным щелчком в браузере. Для такого сценария в проекте есть встроенный резервный набор данных: интерфейс и игры будут работать даже без локального сервера.

### Вариант 2. Через локальный сервер

Если хочется проверить загрузку JSON-файлов напрямую, можно открыть папку через любой простой локальный сервер. Например:

```bash
python3 -m http.server 8000
```

После этого откройте в браузере адрес `http://localhost:8000/`.

## Как загрузить на GitHub

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него **всё содержимое** папки `career-guidance-game/`.
3. Убедитесь, что `index.html` лежит в корне репозитория.
4. Закоммитьте изменения в ветку `main`.

## Как включить GitHub Pages

1. Откройте репозиторий на GitHub.
2. Зайдите в **Settings → Pages**.
3. В блоке **Build and deployment** выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Сохраните настройки.
5. Через короткое время GitHub Pages опубликует сайт по ссылке репозитория.

## Как использовать режим учителя

1. На главной странице нажмите **«Режим учителя»**.
2. Выберите игру из блока запуска модулей.
3. Используйте:
   - таймер для коротких игровых раундов;
   - счёт команд для командной работы;
   - случайный выбор ученика или команды;
   - кнопку **«Режим проектора»** для крупных заголовков и более свободной композиции.
4. Во время любой игры доступны быстрые кнопки **«Следующее задание»** и **«Начать заново»**.

## Как добавить новые профессии и задания

### Профессии

Откройте файл `data/professions.json` и добавьте новый объект по аналогии с существующими записями:

- `id`
- `name`
- `category`
- `description`
- `skills`
- `tools`
- `tasks`
- `education_path`
- `fun_fact`

### Подсказки для игры «Угадай профессию»

Добавляйте карточки в `data/questions.json`:

- `id`
- `profession_id`
- `answer`
- `hints`
- `fact`
- `discussion_prompt`

### Мифы и правда

Редактируйте `data/myths.json`:

- `statement`
- `answer`
- `explanation`
- `discussion_prompt`

### Профессия по навыкам

Редактируйте `data/skills-matching.json`:

- `skills`
- `profession_id`
- `answer`
- `extra_hint`
- `explanation`
- `discussion_prompt`

### Генератор профессий будущего

Редактируйте `data/future-generator.json`:

- `technologies`
- `domains`
- `roles`
- `title_templates`
- `description_templates`
- `skill_sets`
- `discussion_prompts`

## Особенности реализации

- сайт работает без backend;
- навигация построена на hash-маршрутах, поэтому публикация на GitHub Pages не требует дополнительной настройки;
- данные вынесены в JSON;
- есть базовая offline-friendly поддержка через `manifest.webmanifest` и `service-worker.js`;
- базовое состояние занятия сохраняется в `localStorage`.

## Рекомендуемый сценарий занятия

- **Разминка**: генератор профессий или профессии будущего.
- **Основной блок**: 2–3 раунда «Угадай профессию» или «Правда / миф».
- **Обсуждение**: короткий разбор, какие навыки и интересы стоят за разными ролями.
- **Финал**: выбрать одну новую профессию, о которой хочется узнать больше после занятия.
